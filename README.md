# JSON Repository

This repository contains complete JSON schemas, configs, and templates for various use cases.

## Contents

### Schemas
- `config.json` - Application configuration
- `settings.json` - User settings template
- `package.json` - NPM package manifest

### Templates
- `tsconfig.json` - TypeScript configuration
- `tailwind.config.js` - Tailwind CSS config

### Data
- `defaults.json` - Default values
- `schema.json` - JSON schema example
